## Связи

- **UC-02 extend UC-01**: фильтр «Сапсан» применим только после выполнения базового поиска ж/д.
- **UC-03 extend UC-01**: переключение даты в date-strip возможно только на странице результатов поиска.
- **UC-05 extend UC-04**: фильтр «С багажом» применим только на странице результатов поиска авиа.
- **UC-12** — самостоятельный негативный сценарий, проверяющий валидацию формы поиска авиа.
